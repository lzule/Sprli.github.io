2、基础模块的学习
2.1 按键K1控制LED的颜色

  • 具体功能：
    使用K1按键改变LED灯的颜色，并通过led屏幕显示出当前的发光模式

  • 相关函数：
    主要使用了ybrgb库中的设置LED显示函数set(x, x, x)以及ybkey库中的判断按键是否按下函数is_press()

    • set(red_set, green_set, blue_set)
      • red_set表示红色的控制亮灭，其中1表示亮，0表示灭。green_set和blue_set同理
    • is_press()
      • 返回值为1表示检测到K1按键按下，返回值0表示未检测到K1按键按下

  • 具体代码：

    # Untitled - By: HP - 周日 8月 20 2023

import time, lcd, image
from modules import ybkey, ybrgb

lcd.init()
lcd.clear(lcd.WHITE)
RGB = ybrgb()
Key = ybkey()
mode = 0
while True:
    if Key.is_press():
        time.sleep_ms(20)
        while Key.is_press():
            pass
        mode += 1
        mode %= 8 
    if mode == 0:
        RGB.set(0, 0, 0)
    elif mode == 1:
        RGB.set(0, 0, 1)
    elif mode == 2:
        RGB.set(0, 1, 0)
    elif mode == 3:
        RGB.set(0, 1, 1)
    elif mode == 4:
        RGB.set(1, 0, 0)
    elif mode == 5:
        RGB.set(1, 0, 1)
    elif mode == 6:
        RGB.set(1, 1, 0)
    elif mode == 7:
        RGB.set(1, 1, 1)
    
    lcd.draw_string(140, 120, "Mode:" + str(mode), lcd.BLACK, lcd.WHITE)
2.2 使用定时器中断运行函数

  • 具体功能：
    使用定时器中断来处理事件on_timer函数

  • 相关函数：
    主要使用到了Timer库中的Timer定时器配置函数

    • Timer(id, channel, mode=Timer.MODE_ONE_SHOT, period=1000, unit=Timer.UNIT_MS, callback=None, arg=None, start=True, priority=1, div=0)
      • id : Timer ID, [0~2] (Timer.TIMER0~TIMER2)
      • channel : Timer 通道, [Timer.CHANNEL0~Timer.CHANNEL3]
      • mode : Timer 模式, MODE_ONE_SHOT 或者 MODE_PERIODIC 或者 MODE_PWM
      • period : Timer 周期, 在启动定时器后 period 时间， 回调函数将会被调用，(0,~)
      • unit : 设置周期的单位，默认位毫秒（ ms ），Timer.UNIT_S 或者 Timer.UNIT_MS 或者 Timer.UNIT_US 或者 Timer.UNIT_NS
      • callback : 定时器回调函数， 定义了两个参数， 一个是定时器对象 Timer ， 第二个是在定义对象时希望传的参数 arg ，更多请看 arg 参数解释
      • arg : 希望传给回调函数的参数，作为回调函数的第二个参数
      • start : 是否在对象构建成功后立即开始定时器， True ：立即开始； False :不立即开启，需要调用 start() 函数来启动定时器
      • priority : 硬件定时器中断优先级， 与特定的CPU相关， 在K210中，取值范围是[1,7]， 值越小优先级越高
      • div : 硬件定时器分频器，取值范围[0,255]， 默认为0， clk_timer（定时器时钟频率） = clk_pll0（锁相环0频率）/2^(div+1)
    • on_timer(timer)
      • 该函数可以理解为定时器中断函数，即每隔一确定时间该函数就会执行一次

  • 具体代码：

    # Untitled - By: HP - 周日 8月 20 2023


from machine import Timer
import time

def on_timer(timer):
    print("This is on_timer callback")


timer = Timer(Timer.TIMER0, Timer.CHANNEL0,
            mode=Timer.MODE_PERIODIC, period=100,
            unit=Timer.UNIT_MS, callback=on_timer, arg=None)

while True:
    pass
2.3 使用PWM控制呼吸灯

  • 具体功能：
    使用定时器的PWM模式来实现呼吸灯的效果

  • 相关函数：
    主要使用到了machine库中的Timer定时器配置函数和PWM的配置函数

    • Timer(Timer.TIMER0, Timer.CHANNEL0, mode=Timer.MODE_PWM)
      • Timer.MODE_PWM：这里主要使用了定时器0的通道0作为PWM输出模型
      • 其余的参数不需要配置
    • PWM(tim, freq, duty, pin, enable=True)
      • tim : 每个PWM依赖一个定时器来产生波形， 所以这里需要传一个定时器对象，这个定时器对象必须初始化时必须指定定时器 ID 和通道号
      • freq ： PWM 波形频率
      • duty ： PWM 占空比， 指高电平占整个周期的百分比，取值：[0,100]
      • pin ： PWM 输出引脚。 实际物理引脚IO编号。
      • enable ： 是否立即开始产生波形，默认位 True ，及对象生成后立即开始在指定的引脚上产生PWM 波形

  • 具体代码：

    # Untitled - By: HP - 周一 8月 21 2023

from machine import Timer, PWM
from modules import ybrgb, ybkey
import time

# RED=27, GREEN=26, BLUE=29
# LED_PIN = 26
RGB = ybrgb()
RGB.set(0, 0, 0)
mode = 0
tim1 = Timer(Timer.TIMER0, Timer.CHANNEL0, mode=Timer.MODE_PWM)
tim2 = Timer(Timer.TIMER0, Timer.CHANNEL1, mode=Timer.MODE_PWM)
tim3 = Timer(Timer.TIMER0, Timer.CHANNEL2, mode=Timer.MODE_PWM)
pwm1 = PWM(tim1, freq=500000, duty=0, pin=27)
pwm2 = PWM(tim2, freq=500000, duty=0, pin=26)
pwm3 = PWM(tim3, freq=500000, duty=0, pin=29)
duty_Red = 0
duty_Green = 0
duty_Blue = 0
while True:
    pwm1.duty(duty_Red)
    pwm2.duty(duty_Green)
    pwm3.duty(duty_Blue)
    duty_Red += 5
    duty_Red %= 100
    duty_Green += 5
    duty_Green %= 100
    duty_Blue += 5
    duty_Blue %= 100
    time.sleep(0.08)

  • 注意事项：
    K210的LED灯内部有三个小灯，颜色分别为红、绿、蓝，在每次使用LED时，首先使用RGB.set(0, 0, 0)对其进行初始化，然后再进行PWM控制一个灯的亮度。因为如果我前一次测试的时绿灯引脚的PWM，这一次我要测试红灯引脚的PWN，如果不对灯进行set(0, 0, 0)的话，那么上一次绿灯仍是亮的，发光颜色就会收到干扰。

2.4 使用触摸屏

  • 具体功能：
    可以使用手指触摸屏幕来实现绘制图画。

  • 相关函数：
    主要使用到了touchscreen库中的read获取触摸屏状态及坐标函数

    • (status, x, y) = ts.read()
      • status：指当前触摸屏的状态，取值有三种：STATUS_RELEASE（等于1）表示当前未触摸屏幕；STATUS_PRESS（等于2）表示当前只是点击了一下屏幕；STATUS_MOVE（等于3）表示当前持续滑动着屏幕
      • x， y：表示当前触摸点所在屏幕上的坐标
    • ts.STATUS_IDLE：该值为0

  • 具体代码：

    # Untitled - By: HP - 周一 8月 21 2023

import touchscreen as ts
import lcd, image
import time

lcd.init()
ts.init()

img = image.Image()  # 默认读取的是一张黑色图片
#print(img)
img.draw_string(40, 0, "Please touch the screen", color=(0, 0, 255), scale=2)

status_last = ts.STATUS_IDLE
print(status_last)
x_last = 0
y_last = 0

while True:
    (status, x, y) = ts.read()  # 获取触摸屏的状态
    if status_last != status:
        print(status, x, y)     # 如果状态发生了变化，则输出
        status_last = status

    if status == ts.STATUS_MOVE:  # 如果是连续滑动模式，则绘制直线
        print(str(status) + ":line")
        img.draw_line(x_last, y_last, x, y)
    elif status == ts.STATUS_PRESS:  # 如果是点击模式，则绘制一点
        print(str(status) + ":point")
        img.draw_line(x, y, x, y)
    lcd.display(img)
    x_last = x
    y_last = y
2.5 打开摄像头并显示

  • 具体功能：
    打开K210摄像头并将画面显示到lcd显示屏上，在左上角显示出当前的帧数

  • 相关函数：
    主要使用到了sensor库中的摄像头配置与画面获取等函数和lcd库中的显示屏初始化以及显示函数

    • sensor.set_pixformat(format)
      • 用于设置摄像头的输出格式，参数format表示帧格式，可选帧格式有sensor.RGB565、sensor.GRAYSCALE、sensor.YUV422
    • sensor.set_framesize(framesize)
      • 用于设置摄像头输出帧大小，K210最大支持VGA格式，大于VGA将无法获取图像，framesize表示帧大小，可选参数有sensor.VGA(640480)、sensor.QVGA(320240)、sensor.QQVGA(160120)。由于屏幕分辨率大小为320240，故QVGA最为合适
    • sensor.run(enable)
      • 用于图像捕捉功能控制，参数enable：1表示开始抓取图像，0表示停止抓取图像
    • sensor.skip_frames(n, [, time])
      • 表示跳过指定帧或者跳过指定时间内的图像，让相机图像在改变相机设置后稳定下来。参数n：表示跳过n帧图像；参数time：表示跳过指定时间，单位为ms，若n与time皆未指定，该方法跳过300ms的帧；若二者皆指定，该方法会跳过n数量的帧，但将在time ms后返回
    • clock = time.clock() … clock.tick() … clock.fps()
      • 上述三个操作用来计算当前的帧率fps

  • 具体代码：

    # Untitled - By: HP - 周一 8月 21 2023

import sensor, lcd
import time

lcd.init()
sensor.reset()

sensor.set_pixformat(sensor.RGB565)
sensor.set_framesize(sensor.QVGA)
sensor.run(1)
sensor.skip_frames(10)

clock = time.clock()

while(True):
    clock.tick()
    img = sensor.snapshot()
    fps = clock.fps()
    img.draw_string(0, 0, "%2dfps" %(fps), color=(0, 0, 255), scale=2.0)
    lcd.display(img)
    print(fps)
